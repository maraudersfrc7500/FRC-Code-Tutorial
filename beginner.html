Getting Started

There are a couple of things you must first do in order to code for FRC:

1. Install Visual Studio Code

FRC Code is built in a special version of VSCode, hence why it is necessary for it to be installed

https://code.visualstudio.com/download

2. Install WPILib

WPILib is the central library used in FRC coding. As such, it is practically necessary to have in order to code for FRC

https://docs.wpilib.org/en/stable/docs/zero-to-robot/step-2/wpilib-setup.html

3. Install FRC Driver Station and Other Tools

FRC uses a laptop as the communicator between the people and the robot. Therefore, the FRC Game Tools must be installed as well

https://docs.wpilib.org/en/stable/docs/zero-to-robot/step-2/frc-game-tools.html
